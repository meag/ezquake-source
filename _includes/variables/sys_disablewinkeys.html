sys_disableWinKeys
Type:
boolean
Remarks:
Note: Windows keys are now bindable (LWINKEY, RWINKEY, WINKEY, POPUPMENU). Obviously only useful on NT/2K/XP with sys_disableWinKeys 1 (and possibely useful in Linux too!).
ValueDescription
falseEnable.
trueDisable winkeys from alt-tabbing you (only works in NT/2K/XP, and only when the client is in focus).
